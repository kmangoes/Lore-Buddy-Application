# Lore Buddy Application
Full-stack writing workspace geared towards creating in-depth character studies and world building. 

## App Functions

### 1. User 
- User creates a profile, which holds the entirety of their workspace. Creates a profile with an email and password, and is then redirected to login afterwards.
- User creates 'Worldspaces' for each universe/loreset they create. **Family Trees** (and thus, **Character Sheets**), **Environments**, **Timelines**, and written stories all fall under here.

### 2. Provider 
- Primarily oversees general user accounts statistics. Can delete accounts. 
- Can view the number of total accounts, when accounts were created, activity level, etc.
